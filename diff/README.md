# mdxNuxt Diff

This repository contains a Go module for diffing functionality.

## Build

```sh
go build ./...
```

## Test

```sh
go test ./...
```

## Additional Commands

- Run tests with coverage:

  ```sh
  go test -cover ./...
  ```

- Format code:

  ```sh
  go fmt ./...
  ```

## Requirements

- Go 1.18+
- Clone this repo and run commands from the root directory.
